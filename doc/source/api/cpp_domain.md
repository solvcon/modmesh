# Domain visualizer

The 2/3D visualizer (see {doc}`../pilot/domain`) renders through
[QRhi](https://doc.qt.io/qt-6/qrhi.html), Qt's portable graphics abstraction.

{cpp:class}`solvcon::RDomainWidget` is the Python-facing control object.
